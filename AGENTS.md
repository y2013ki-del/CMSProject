# AGENTS — 문서 진입 가이드

이 프로젝트에서 작업을 시작할 때는 아래 순서를 따릅니다.

## 기본 순서

1. `README.md`로 프로젝트 개요와 문서 인덱스를 확인
2. `md file/DOC_INDEX.md`에서 현재 작업 유형에 맞는 문서를 선택
3. 선택된 하위 문서만 추가로 읽고 작업
4. 장애/오류 대응이면 `md file/ERROR_LOG.md`를 우선 확인
5. 최근 변경 맥락이 중요하면 `md file/update.md`를 확인

## 원칙

- `README.md`는 요약과 링크만 유지
- 상세 내용은 각 하위 md에만 기록
- 새 내용을 추가할 때는 기존 문서를 키우기보다 맞는 대분류 md에 기록
- 새로운 대분류가 생기면 `md file/DOC_INDEX.md`에도 라우팅 규칙을 추가
- 기능/구조/운영 흐름이 바뀐 작업은 커밋 전에 `md file/update.md`에 날짜 기준으로 추가
- `md file/update.md`에는 세부 설계 전체가 아니라 “이번 버전에 무엇이 바뀌었는지”만 요약
